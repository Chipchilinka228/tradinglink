ИНСТРУКЦИЯ

1. Откройте index.html двойным кликом.
2. Чтобы форма отправляла заявки:
   - зарегистрируйтесь на https://formspree.io
   - создайте форму
   - замените:
     your_form_id
     на ваш ID формы в index.html

3. Для загрузки сайта:
   - Beget
   - Timeweb
   - Reg.ru
   - SprintHost

4. Загрузите все файлы из папки на хостинг.

5. SEO уже добавлено:
   - title
   - description
   - keywords
   - адаптивность
   - правильная структура заголовков
